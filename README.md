Pizza ordering test assignment made with React, Redux, Bootstrap on frontend and node.js with Fastify on backend.

Demo: https://pizza-task-valerr.herokuapp.com/

Install
-------

~~~
   $make install
   $make start
   open http://localhost:5000
~~~
